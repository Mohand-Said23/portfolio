Site web et application pour un restaurant

Description :
Dans ce projet, j'ai conçu un site web et une application mobile pour un restaurant de pizzas, en utilisant Django et Python pour une gestion dynamique et facile du menu via une interface d'administration sur-mesure. L'application mobile développée avec Kivy permet aux clients de passer des commandes de manière efficace.

Points Forts :
- Interface d'administration personnalisée : Permet une mise à jour facile et rapide du menu.

- Application mobile cross-platform : Assure une expérience utilisateur uniforme sur les appareils iOS et Android.

- Déploiement sur Heroku : Offre une haute disponibilité et facilité de maintenance.

Impact : Le projet a optimisé l'interaction client et amélioré la gestion opérationnelle du restaurant, tout en me permettant de renforcer mes compétences en développement web et mobile.

Technologies utilisées: HTML, CSS, DJANGO, PYTHON, KIVY, HEROKU, HTTP et JSON.
