Application Mobile "AI"

Ce projet personnel offre aux utilisateurs d'iPhone ou d'Android la possibilité de prendre une photo et d'afficher "l'âge estimé" grâce à un algorithme de machine learning.

L'application se connecte à une API du service Azure Cognitive de Microsoft.

Technologies utilisées: Xamarin.Forms, C#, .NET, Azure Cognitive Services, HTTP et JSON.
